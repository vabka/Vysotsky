# Стек
В проекте используется:
- Linq2DB
- PostgreSQL
- ASP NET Core (.NET 5)
- Fluentmigrator
- Docker + Docker-compose
- nuke.build
- Grafana / Vector / Loki
- xUnit
- FluentAssertions
- Scrutor + MS.DI
- Serilog
- Flurl
- NSwag
